## Tại sao lại cần Excel Addin?

Vì dự án cần một tính năng để xây dựng Pivot Report. Nhưng mà tính năng đó xử dụng trên web quá nặng và lag. Dẫn đến việc chúng tôi quyết định chuyển sang Excel Addin. Để pull toàn bộ data về và user có thể tự xây dựng Pivot Report mà không cần phải chờ đợi.

## Những lưu ý khi làm việc với Excel Addin

- Không tải bản excel từ apple store mà tải bản cài từ trang chủ của Microsoft.
- Để debug được thì cần phải chạy lệnh sau 

```
defaults write com.microsoft.Excel OfficeWebAddinDeveloperExtras -bool true
```

- Vì excel Addin build ra web nên khi code cần cẩn thận các feature có thể không được support ở các browser cũ.
- Excel Addin chỉ call được api từ https nên cần phải chuyển hết các api về https (có thể dùng ngrok để dev)
